# Supernova Pro

A Minecraft 1.20.4 Fabric hacked client for Crystal PvP.

![Screenshot](screenshot.png)

## Features

### Combat
- **AutoCrystal** - Automated end crystal placement and breaking
- **AutoCrystalBase** - Base placement helper for crystal combat
- **AutoTrap** - Trap enemies with obsidian
- **AutoWeb** - Place cobwebs on enemies
- **AutoLadder** - Place ladders near enemies
- **AutoCity** - Break opponent's obsidian
- **AutoAnchor** - Respawn anchor combat
- **AutoHoleFill** - Fill holes automatically
- **PistonCrystal** - Piston-based crystal setups
- **BedAura** - Bed placement combat
- **KillAura** - Melee combat automation
- **Surround** - Self-trapping with obsidian
- **Burrow** - Hide inside blocks

### Player
- **AutoGapple** - Auto eat golden apples
- **AutoArmor** - Auto equip armor
- **AutoTool** - Auto switch tools
- **AutoMine** - Auto mining
- **AutoPot** - Auto splash potions
- **AutoHeal** - Auto exp bottles
- **AutoTrade** - Auto villager trading
- **AutoPearl** - Auto ender pearl throw
- **Freecam** - Free camera movement
- **PacketMine** - Fast mining
- **TimerModule** - Game speed control

### Movement
- **Speed** - Movement speed boost
- **Fly** - Flight
- **Scaffold** - Auto bridge
- **Step** - Step height increase
- **Velocity** - Knockback reduction
- **Sprint** - Auto sprint
- **HoleSnap** - Teleport to holes

### Render
- **ESP** - Entity highlighting
- **HoleESP** - Hole visualization
- **Tracers** - Entity tracers
- **NameTags** - Enhanced name tags
- **CrystalChams** - Crystal chams
- **XRay** - Block ESP
- **Shader** - Post-processing effects

### Exploit
- **Blink** - Packet freeze
- **PearlPhase** - Phase through blocks
- **WallClip** - Clip through walls
- **XCarry** - Carry items in containers

## Requirements

- Minecraft 1.20.4
- Fabric Loader 0.15.7+
- Java 17+

## Download

Download the latest release from [Releases](https://github.com/bhirapth/Supernova-Pro/releases).

## Build

```bash
git clone https://github.com/bhirapth/Supernova-Pro.git
cd Supernova-Pro
./gradlew build
```

The built jar will be in `build/libs/`.
