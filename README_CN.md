# Supernova Pro

Minecraft 1.20.4 Fabric 水晶PvP客户端。

![截图](screenshot.png)

## 功能

### 战斗
- **AutoCrystal** - 自动水晶放置与引爆
- **AutoCrystalBase** - 水晶底座放置辅助
- **AutoTrap** - 黑曜石困住敌人
- **AutoWeb** - 蜘蛛网陷阱
- **AutoLadder** - 自动放置梯子
- **AutoCity** - 挖掘敌人黑曜石
- **AutoAnchor** - 重生锚战斗
- **AutoHoleFill** - 自动填坑
- **PistonCrystal** - 活塞水晶
- **BedAura** - 床战斗
- **KillAura** - 近战自动攻击
- **Surround** - 自我黑曜石围困
- **Burrow** - 躲入方块

### 玩家
- **AutoGapple** - 自动吃金苹果
- **AutoArmor** - 自动装备盔甲
- **AutoTool** - 自动切换工具
- **AutoMine** - 自动挖矿
- **AutoPot** - 自动药水
- **AutoHeal** - 自动经验瓶
- **AutoTrade** - 自动村民交易
- **AutoPearl** - 自动末影珍珠
- **Freecam** - 自由视角
- **PacketMine** - 快速挖掘
- **TimerModule** - 游戏加速

### 移动
- **Speed** - 移动加速
- **Fly** - 飞行
- **Scaffold** - 自动搭桥
- **Step** - 跨越高度
- **Velocity** - 击退减缓
- **Sprint** - 自动疾跑
- **HoleSnap** - 传送至安全坑

### 渲染
- **ESP** - 实体高亮
- **HoleESP** - 安全坑显示
- **Tracers** - 实体追踪线
- **NameTags** - 增强名牌
- **CrystalChams** - 水晶透视
- **XRay** - 方块透视
- **Shader** - 后处理特效

### 漏洞利用
- **Blink** - 数据包冻结
- **PearlPhase** - 穿墙珍珠
- **WallClip** - 穿墙
- **XCarry** - 手持容器物品

## 环境要求

- Minecraft 1.20.4
- Fabric Loader 0.15.7+
- Java 17+

## 下载

从 [Releases](https://github.com/bhirapth/Supernova-Pro/releases) 下载最新版本。

## 构建

```bash
git clone https://github.com/bhirapth/Supernova-Pro.git
cd Supernova-Pro
./gradlew build
```

构建产物位于 `build/libs/`。
